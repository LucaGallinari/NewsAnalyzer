About

News Analyzer is a cloud-based application that allows users to obtain more knowledge about a specific news (analyze page), by showing different informations of the main entities found in it. It also works as a news provider (homepage) and manager (if logged)
An authenticated user may access these additional functions:
  • Manage favorite news
  • Manage personal news filters (keywords)
  • Daily receive an email with news regarding a filter
  • Look at your analysis history
  • Easily share news on Google+
Source: https://github.com/LucaGallinari/NewsAnalyzer

    List users by entity
    Description Given an entity as argument this API return the list (in JSON) of logged users that extracted it today.
    Path /api/analyze/list_users_by_entity
    Arguments entity=[name], name of the entity (UTF-8)
    Response [(email1),(email2),(email3),..]
    Example(s) /api/analyze/list_users_by_entity?entity=Barack+Obama
